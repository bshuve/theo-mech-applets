In this second page, y1 and y5 have been set to 0. Try moving the sliders for y2, y3, and y4. The gray line represents the local classical path between points y1 and y5.

You can see that the the values for y2, y3, and y4 that minimize the action is not the global classical trajectory from the boundary endpoints, but it is the local classical trajectory between points y1 and y5. Changing y2, y3, and y4 allow us to locally minimize the action to find the classical path subject to a set of boundary conditions that differ from the original ones provided. This action is still larger than the global classical path (the black line global classical trajectory has a minimum action of -1334), but it is the minimum for the local set conditions.
