Free-fall projectile in 1D: Page 1/4

(Chrome Recommended.)

In this applet, you are going to be looking at the relationship between minimizing the action globally vs. locally. The system is uniform acceleration due to gravity. However, instead of using parametrized curves, you will explicitly specify the vertical position of the particle at various points on its trajectory.

The black curve in the plots represents the classical path globally (minimizing the action between the boundary points of \(t=0\) and \(t=20\) s). The white curve minimizes the "local" action defined only between the two points it connects, so it shows the classical path locally. In this first page, \(y_2\), \(y_3\), \(y_4\), and \(y_5\) are set at their respective points that follow the global classical trajectory. Using the slider, \(y_1\) can be varied. Try moving the slider to see how varying one point can affect the action value.





The action is minimized when \(y_1\) is 51, and falls on the global classical trajectory path. By putting \(y_1\) at 51, we are minimizing the actions locally between each of the points, and consequently we are minimizing the global action as well.


The Global Action graph to the right shows the action as a function of the indicated \(y_i\), assuming that all other points are fixed. Here, it shows the action as a function of \(y_1\), assuming that \(y_2\), \(y_3\), \(y_4\), and \(y_5\) don't change. The "Randomize" button moves \(y_1\) to a random value, while "Classical Path" sets it to the value minimizing the global action.
