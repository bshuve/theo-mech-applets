Global and Local Action Minimization

In this applet, you will look at the relationship between minimizing the action globally vs. locally when varying a path with fixed endpoints: \(y(t_i)=y(t_f)=0\) for \(t_i=0\) and \(t_f=20\) s. We consider a point particle undergoing uniform acceleration due to gravity. We set \(g=2\,\,\mathrm{m/s^2}\) to simplify the calculations.

You will vary the path by fixing values of \(y(t)\) for times in between the endpoints.

The black curve in the plots represents the classical path globally (minimizing the action between the boundary points of \(t=0\) and \(t=20\) s). The white curve minimizes the "local" action defined only between the two points it connects, so it shows the classical path locally. In this first page, \(y_2\), \(y_3\), \(y_4\), and \(y_5\) are set at their respective points that follow the global classical trajectory. Using the slider, \(y_1\) can be varied. Try moving the slider to see how varying one point can affect the action.





The action is minimized when \(y_1\) is 51 m, and falls on the global classical trajectory path. By putting \(y_1\) at 51 m, we are minimizing the action locally between the beginning and \(y_2\), and consequently we are minimizing the global action as well.


Note: The "Randomize" button moves \(y_1\) to a random value, while "Classical Path" sets it to the value minimizing the global action.
