The action is minimized when y1 is . In order to find the y1 that minimizes local action, we must follow the classical path locally. Since we are minimizing the action locally between points y0 and y2, we are able to minimize the action of the entire path for the constrained y2, y3, y4, and y5 points. This minimizes our global action for these constrained points, although we are unable to achieve the global minimum action, because that would require minimizing the action throughout the whole path as shown through the black path (which has an action of -1334).
