Free-fall projectile in 1D: Page 2/4

In this applet, you are going to be looking at global vs. local action. The black line in the plots represents the classical path globally. The white one minimizes the local action between the two points it connects, so it shows the classical path locally between those two points. In this first page, y2, y3, y4, and y5 are randomized each time. Using the slider, y1 can be varied. Try moving the slider to see how varying one point can affect the action value.





The action is minimized when y1 is . This shows that in order to minimize the action, we must follow the classical path locally. Since we are minimizing the actions locally between points y0 and y2, we are able to minimize the action of the entire path, for the constrained y2, y3, y4, and y5 points.

The Integral of Energy graph to the right also shows the action as a function of the given y, assuming that all other points are fixed. For example, plot 1 shows the action as a function of y1, assuming that y2, y3, y4, and y5 don't change. You can click the checkboxes to look at how other plots are affected by changing the y1 value with the slider.
