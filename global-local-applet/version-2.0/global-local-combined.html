Welcome to Action Minimization!

Goals of this applet:

  • Show how trajectories minimize action globally and locally
  • Show the relationship between classical paths and action minimization
  • Show constraints affect the minimization process

Journey through this page: There are 4 interactive applets with increasing complexity.

Global and Local Action Minimization

An Interactive Exploration of Mechanics

Sections

Vary One Point Fixed Random Points Multiple Variables Full Control

Section 1: Varying a Single Point

In this first section, you'll explore how changing a single trajectory point \(y_1\) affects the action. Compare the blue parameterized path (which must pass through your chosen point) with the black classical path (the global minimum).

Things to do:

  • Adjust the slider to adjust parameterized path.
  • Notice how action depends on trajectory shape.
  • Observe where action is minimized

Section 2: Constraints and Local Minimization

Now the scenario changes! Points \(y_2\), \(y_3\), \(y_4\), and \(y_5\) are fixed to random values. Only \(y_1\) remains adjustable. Note: even with constraints, you can minimize action locally by choosing the optimal \(y_1\) between the fixed endpoints.

Things to do:

  • Adjust \(y_1\) and find point where action is minimized - classical path!
  • Observe local vs. global minimum

Section 3: Multiple Variables with Endpoint Constraints

Here, the endpoints \(y_1 = y_5 = 0\) are fixed, but you control three intermediate points: \(y_2\), \(y_3\), and \(y_4\). Notice the two different trajectory curves: the black global classical path and the blue parameterized path. Try to find the constrained minimum action?

To do:

  • Minimize action with multiple free points
  • Observe changes in action plots of the point that is adjacent to the point you move and one that is not.

Section 4: Full Control – Finding the Global Minimum

You now have complete control over all five trajectory points. Use the action plots for each variable (toggle them with checkboxes) to systematically find the global classical trajectory that minimizes action. The target action is \(-1334\) J.s. Can you find it?

To do:

  • Find the global minimum by minimizing each local minimum!
