Coordinate transformations for a simple system

We start by considering two different coordinate choices for a familiar system: projectile motion under constant acceleration due to gravity. While Cartesian coordinates work well for this system, we can gain insight into coordinate transformations by considering the motion using an alternative coordinate choice that's easy to understand.

Consider an object of mass \(m\) experiencing constant acceleration \(-g\) in the vertical direction. The object is launched from height \(h\) with horizontal motion but no initial vertical velocity. In Cartesian coordinates (\(y\)), we know the trajectory follows: \[ y(t) = -\frac{1}{2}gt^2 + h \] To explore coordinate transformations, we'll analyze the same physical system using a non-Cartesian coordinate, \(w = y^2\). While this coordinate choice isn't physically motivated, it provides a mathematically straightforward example of how coordinate transformations affect our description of motion.

We can think of the change from y to w as a transformation of the vertical axis. As a result, the trajectory w(t) looks different from y(t). To see this, we animate a particular trajectory, showing both y(t) and w(t).



\(y\) can take on any real value, whereas \(w\ge0\). Therefore \(w\) unambiguously gives the same path as \(y\) for \(y\ge0\).
