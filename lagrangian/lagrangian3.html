Generalized Coordinate Lagrangians: Page 3

Non-classical Path

Consider an object of mass \(m=1.0\) kg constrained to move in the \(y\)-direction under gravitational acceleration \(-g\). The kinetic energy of the object is \(T=\frac{1}{2}m\dot{y}^2\) and the potential energy is \(U=mgy\).

We now consider deviations from the classical path. We consider a family of paths defined by \(y(t) = 1-t^p\) for some parameter \(p\) (taking \(g\) to be 2 for simplicity). We can also study this family path using a different coordinate system \(w\), where \(w(y) = y^2\) so \(w(t) = (1-t^p)^2\). Try moving the sliders and observing how the motion and graphs change.




We know that \(\frac{\partial L}{\partial w} = \frac{d}{dt}(\frac{\partial L}{\partial \dot{w}})\) and that \(L = T - U\).
So we know that \(\frac{\partial L}{\partial w} = \frac{\partial T}{\partial w} - \frac{\partial U}{\partial w}\) and \(\frac{d}{dt}(\frac{\partial L}{\partial \dot{w}}) = \frac{d}{dt}(\frac{\partial T}{\partial \dot{w}} - \frac{\partial U}{\partial \dot{w}})\).
\(T = \frac{1}{2}m(\frac{\dot{w}}{2 \sqrt{w}})^2\) since \(\dot{y} = \frac{\dot{w}}{2 \sqrt{w}}\) and \(U = 2m\sqrt{w}\) since \(y = \sqrt{w}\) and \(g = 2\).
Knowing this, we get:
\(\frac{\partial}{\partial w}(\frac{1}{2}m\frac{\dot{w}^2}{4w}) - \frac{\partial}{\partial w}(2m \sqrt{w}) = \frac{d}{dt}(\frac{\partial}{\partial \dot{w}}(\frac{1}{2}m \frac{\dot{w}^2}{4w}) - \frac{\partial}{\partial \dot{w}}(2m \sqrt{w}))\)
Which simplifies into:
\(-\frac{1}{2}\dot{w}^2 - 4w^{3/2} = \ddot{w}w - \dot{w}^2\)

Recall that for the \(w\) parameterizations, the following apply:
\(w = (1-t^p)^2\)
\(\dot{w} = 2y\dot{y} = 2(1-t^p)(-pt^{p-1})\)
\(\ddot{w} = 2(\dot{y}^2 + y\ddot{y}) = 2((-pt^{p-1})^2 + (1-t^p)(-p(p-1)t^{p-2}))\)

When we plug these into the equation above, we get the left-hand side of the equation as \(-\frac{1}{2}(2(1-t^p)(-pt^{p-1}))^2 - 4((1-t^p)^2)^{3/2}\), which expands to \(-2p^2t^{2p-2} + 4p^2t^{3p-2} - 2p^2t^{4p-2} - 4 + 12t^p - 12t^{2p} + 4t^{3p}\).
The right-hand side is \(2((-pt^{p-1})^2 + (1-t^p)(-p(p-1)t^{p-2}))(1-t^p)^2 - (2(1-t^p)(-pt^{p-1}))^2\) and expands to \(-2p^2t^{2p-2} - 2p^2t^{3p-2} - 2p^2t^{p-2} + 2pt^{p-2} + 6p^2t^{2p-2} - 6pt^{2p-2} + 6pt^{3p-2} - 2pt^{4p-2}\).
The correct p value satisfies this equation for all t values between 0 and 1. Here, the two sides are only equal when \(p=2\). You can check this by graphing each side in Desmos and using a slider for \(p\)!
Corrected Derivation General Equation of Motion (from Page 2):
From the general derivation in the \(w\) coordinate system, we have: \[ 2w \ddot{w} - \dot{w}^2 + 4g w^{3/2} = 0 \] Substituting \(g = 2 m/s^2\), we get: \[ 2w \ddot{w} - \dot{w}^2 + 8 w^{3/2} = 0 \] Using the definition of w(t), we can derive \(w(t), \dot{w},\) and \(\ddot{w}\) as follows: \[ w(t) = (1 - t^p)^2 \] \[ \dot{w}(t) = \frac{d}{dt}(1 - t^p)^2 = 2(1 - t^p)(-p t^{p-1}) = -2p t^{p-1} (1 - t^p) \] \[ \ddot{w}(t) = \frac{d}{dt}[-2p t^{p-1} (1 - t^p)] = -2p [(p-1)t^{p-2}(1-t^p) - p t^{2p-2}] \] We can thus substitute these values into the LHS as follows: \[ \begin{align*} \text{LHS} &= 2w \ddot{w} - \dot{w}^2 + 8 w^{3/2}\\ \\ 2w \ddot{w} &= 2(1 - t^p)^2 \cdot [-2p ((p-1)t^{p-2}(1-t^p) - p t^{2p-2})] \\ &= -4p (1 - t^p)^3 (p-1)t^{p-2} + 4p^2 (1 - t^p)^2 t^{2p-2} \\ \\ -\dot{w}^2 &= -[-2p t^{p-1} (1 - t^p)]^2 \\ &= -4p^2 t^{2p-2} (1 - t^p)^2 \\ \\ 8w^{3/2} &= 8[(1 - t^p)^2]^{3/2} = 8|1 - t^p|^3 = 8(1 - t^p)^3 \\ \end{align*} \] Combine All Terms: \[ \begin{align*} \text{LHS} = & \left[ -4p (p-1) (1 - t^p)^3 t^{p-2} \right] \\ + & \left[ 4p^2 t^{2p-2} (1 - t^p)^2 \right] \\ - & \left[ 4p^2 t^{2p-2} (1 - t^p)^2 \right] \\ + & \left[ 8 (1 - t^p)^3 \right] \\ \end{align*} \] Notice the second and third terms cancel each other. This simplifies to: \[ \text{LHS} = (1 - t^p)^3 \left[ -4p(p-1)t^{p-2} + 8 \right] \]
Set Equation to Zero: \[ (1 - t^p)^3 \left[ 8 - 4p(p-1)t^{p-2} \right] = 0 \] Since \((1 - t^p)^3 \neq 0\) for \(t < 1\), we must find the value of p such that: \[ 8 - 4p(p-1)t^{p-2} = 0 \quad \text{for all } t \] Solve for \(p\): This equation must hold for all \(t\), which is only possible if: \[ p = 2 \] When \(p = 2\), \(t^{p-2} = t^{0} = 1\), so: \[ 8 - 4(2)(2-1)(1) = 8 - 8 = 0 \] For other values of \(p\), the expression depends on \(t\) and cannot be equal to zero.
