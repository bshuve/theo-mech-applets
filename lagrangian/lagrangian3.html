Generalized Coordinate Lagrangians: Page 3

Non-classical Path

Consider an object of mass \(m=1.0\) kg constrained to move in the \(y\)-direction under gravitational acceleration \(-g\). The kinetic energy of the object is \(T=\frac{1}{2}m\dot{y}^2\) and the potential energy is \(U=mgy\).

We now consider deviations from the classical path. We consider a family of paths defined by \(y(t) = 1-t^p\) for some parameter \(p\) (taking \(g\) to be 2 for simplicity). We can also study this family path using a different coordinate system \(w\), where \(w(y) = y^2\) so \(w(t) = (1-t^p)^2\). Try moving the sliders and observing how the motion and graphs change.




From the general derivation in the \(w\) coordinate system from page 2, we have: \[ 2w \ddot{w} - \dot{w}^2 + 4g w^{3/2} = 0 \] Substituting \(g = 2 m/s^2\), we get: \[ 2w \ddot{w} - \dot{w}^2 + 8 w^{3/2} = 0 \] Using the definition of w(t), we can derive \(w(t), \dot{w},\) and \(\ddot{w}\) as follows: \[ w(t) = (1 - t^p)^2 \] \[ \dot{w}(t) = \frac{d}{dt}(1 - t^p)^2 = 2(1 - t^p)(-p t^{p-1}) = -2p t^{p-1} (1 - t^p) \] \[ \ddot{w}(t) = \frac{d}{dt}[-2p t^{p-1} (1 - t^p)] = -2p [(p-1)t^{p-2}(1-t^p) - p t^{2p-2}] \] We can thus substitute these values into the LHS as follows: \[ \begin{align*} \text{LHS} &= 2w \ddot{w} - \dot{w}^2 + 8 w^{3/2}\\ \\ 2w \ddot{w} &= 2(1 - t^p)^2 \cdot [-2p ((p-1)t^{p-2}(1-t^p) - p t^{2p-2})] \\ &= -4p (1 - t^p)^3 (p-1)t^{p-2} + 4p^2 (1 - t^p)^2 t^{2p-2} \\ \\ -\dot{w}^2 &= -[-2p t^{p-1} (1 - t^p)]^2 \\ &= -4p^2 t^{2p-2} (1 - t^p)^2 \\ \\ 8w^{3/2} &= 8[(1 - t^p)^2]^{3/2} = 8|1 - t^p|^3 = 8(1 - t^p)^3 \\ \end{align*} \] Combining like terms, we get: \[ \begin{align*} \text{LHS} = & \left[ -4p (p-1) (1 - t^p)^3 t^{p-2} \right] \\ + & \left[ 4p^2 t^{2p-2} (1 - t^p)^2 \right] \\ - & \left[ 4p^2 t^{2p-2} (1 - t^p)^2 \right] \\ + & \left[ 8 (1 - t^p)^3 \right] \\ = & (1 - t^p)^3 \left[ -4p(p-1)t^{p-2} + 8 \right] \end{align*} \] In order to find the values of \(p\) for which the Euler-Lagrange equation is satisfied, we need to find the values such that \[ (1 - t^p)^3 \left[ 8 - 4p(p-1)t^{p-2} \right] = 0. \] Since \((1 - t^p)^3 \neq 0\) for \(t < 1\), we must find the value of p such that: \[ 8 - 4p(p-1)t^{p-2} = 0 \quad \text{for all } t \] This equation must hold for all \(t\), which is only possible if: \[ p = 2 \] When \(p = 2\), \(t^{p-2} = t^{0} = 1\). Substituting, we get: \[ 8 - 4(2)(2-1)(1) = 8 - 8 = 0 \] which confirms the Euler-Lagrange equation holds! This matches the observations that we get from the applet.
