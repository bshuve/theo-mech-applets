Newton's 2nd Law and Least Action

This applet explores the connection between Newton's 2nd Law and the Principle of Least Action, showing graphically that the path minimizing the action is also the only path which satisfies Newton's 2nd Law.

We consider a simple physical system: a point mass moving freely under uniform gravitational acceleration. We consider the motion of the object from \(t_i=-\sqrt{\frac{2y_0}{g}}\) to \(t_f=\sqrt{\frac{2y_0}{g}}\), with its position beginning and ending at \(y=0\), with \(y_0=1\) m. The classical path is \(y(t) = y_0-\frac{1}{2}gt^2\), which has constant acceleration \(\ddot{y}=-g\). This acceleration satisfies Newton's 2nd Law, \(F_y=m\ddot{y}\).

Alternate paths must satisfy the same boundary condition as the classical path. We consider a family of paths \(y(t) = (y_0-\frac{1}{2}gt^2) + py_0\cos(\frac{\pi}{2}\frac{t}{t_i}) \) parametrized by \(p\). This gives the classical path when \(p=0\).

Question 1: Check that the alternate path has the property \(y(t_i)=y(t_f)=0\).



Answer: The acceleration is \(-g-p(\frac{\pi}{2}\frac{1}{t_i})^2\cos(\frac{\pi}{2}\frac{t}{t_i}) \). This acceleration is \(-g\) only when \(p=0\).

LEGEND



The path minimizing the action (\(p = 0\)) is identical to the path that satifies Newton's 2nd Law. This means that real world trajectories conincide with the path where action is minimized and Newton's 2nd Law holds.
